# NAME

YukiWikiMini - Yet Another Wiki Clone.

# DESCRIPTION

YukiWikiMini is a smallest subset of Wiki.

# INSTALL

It can work on Apache CGI or Plack/PSGI

## Apache CGI

put this directory under apache cgi directory.


## Plack/PSGI

```
cpanm Task::Plack
plackup app.psgi
```
